C 語言概覽

C 語言是一種廣泛使用的通用程式語言。高效率、接近硬體，廣泛用於作業系統、嵌入式開發、驅動程式等領域。

C 語言簡史

C 語言由 Dennis Ritchie 於 1972 年在貝爾實驗室開發，用於 Unix 作業系統核心的撰寫。

語法風格與 B 語言與 ALGOL 系譜相關。

基本語法

資料型別

C 語言支援以下基本資料型別：

  • int
  • float
  • double
  • char

控制流程

常見控制結構包括：

  • if, else
  • switch, case
  • while, do-while
  • for

函式定義

int add(int a, int b) {
    return a + b;
}

函式與模組化

C 語言支援將程式分成多個檔案，透過 #include 和 .h 檔案進行模組化開發。

Header 檔案範例

// math_utils.h
int add(int a, int b);
int multiply(int a, int b);

編譯與鏈結

最常見的 C 編譯器為 GNU GCC。基本編譯流程如下：

gcc -c main.c        # 編譯為目標檔
gcc -o app main.o    # 鏈結為可執行檔

多檔案專案

可使用以下命令處理多個原始檔案：

gcc -o app main.c utils.c
